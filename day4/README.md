## To prepare project

Create .env file in root directory with following format: 
```
HTTP_PORT=
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=
DB_HOST=
DB_PORT=
DB_DIALECT=
```

## To run project

```shell
> npm install
> npx sequelize-cli db:migrate
> npm start
```
